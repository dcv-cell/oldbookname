### 2. 配置Git LFS
用于管理图片等大文件，避免Git仓库过大：
```powershell
# 安装Git LFS
git lfs install

# 跟踪图片文件
git lfs track "*.png"
git lfs track "*.jpg"
git lfs track "*.jpeg"
git lfs track "*.gif"
git lfs track "*.webp"

# 验证跟踪配置
git lfs ls-files
```
